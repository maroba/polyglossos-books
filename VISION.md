In diesem Projekt möchte ich Lehrbücher für verschiedene Sprachen erstellen.
Jedes der Lehrbücher soll einen Lernenden von Niveau A0 auf C1 bringen. Das
wird also sehr umfangreich. Die Bücher sollen didaktisch gut aufbereitet sein.

Zu diesem Zweck möchte ich, dass wir gemeinsam verschiedene Custom Agents und
Custom Skills planen. An folgende Custom Agents habe ich gedacht:

teacher: Dieser Agent übernimmt die didaktische Planung und das Schreiben

reviewer: Dieser Agent ist Sprachexperte und Lektor und prüft das Material auf 
Fehler und gibt Verbesserungsvorschläge

reader: Dieser Agent übernimmt die Perspektive des Lernenden und gibt Feedback,
was nicht verständlich ist.

ai-guard: Dieser Agent prüft das Material darauf, dass es nicht klingt, wie von
einer KI generiert sondern von einem echten Menschen.

Weitere Ideen? Lass uns diskutieren!
